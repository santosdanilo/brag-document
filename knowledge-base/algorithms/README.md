# Algorithms Knowledge Base

Index of algorithm problems and DSA patterns organized by topic.

## How to Add Problems

Problems are stored at `solutions/{source-name}/{problem-name}/problem.md`.

- `{source-name}`: lowercase source (e.g., `leet-code`, `hackerrank`)
- `{problem-name}`: `{number}-{name-in-kebab-case}` (e.g., `209-minimum-size-subarray-sum`)
