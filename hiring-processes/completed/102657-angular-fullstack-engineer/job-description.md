# 102657 - Angular Fullstack Engineer

**Location:** Multiple Countries
**Type:** Full Time
**Level:** Manager/Supervisor

## Summary

We are seeking a senior Angular Fullstack Engineer to lead stabilization, modernization, and delivery of an existing production Angular application. This role focuses on Angular (v18+) best practices, strong TypeScript engineering, and practical full-stack troubleshooting with Node.js and REST APIs. The position is ideal for an engineer who prioritizes reducing technical debt, improving reliability, and enabling safe, continuous delivery in a pressured production environment.

You will work across front-end architecture, backend integration, testing, and operational stability—using AI-assisted tools as part of a disciplined engineering workflow—to increase maintainability, observability, and deployment confidence.

## Responsibilities

- Design and implement robust Angular (v18+) solutions using standalone components, signals/reactivity, and RxJS.
- Modernize legacy Angular approaches and refactor anti-patterns safely in production systems.
- Develop strongly typed, modular TypeScript code focused on readability and long-term maintainability.
- Integrate and troubleshoot REST API contracts and data flow issues with backend services.
- Support and enhance lightweight Node.js middleware/services to resolve integration problems.
- Build automated acceptance/regression tests to improve deployment confidence and prevent recurring defects.
- Proactively address security and dependency issues, updating libraries and fixing vulnerabilities safely.
- Investigate and resolve production incidents; improve observability and monitoring to reduce operational risk.
- Collaborate with platform and DevOps teams on CI/CD, deployment troubleshooting, and environment configuration.
- Mentor junior developers through code review, pairing, and clear communication of technical tradeoffs.

## Requirements

- 5+ years of experience building and maintaining Angular applications, including deep knowledge of Angular v18+ concepts: standalone components, signals/reactivity, and RxJS.
- Strong TypeScript skills with a focus on type safety, modularity, and maintainable code.
- Practical experience integrating with REST APIs and troubleshooting API/data flow issues.
- Hands-on experience with Node.js to support middleware/services and debug backend integration problems.
- Experience modernizing legacy front-end code and safely refactoring production systems to reduce technical debt.
- Experience building automated acceptance or regression tests to improve quality and deployment confidence.
- Proven track record of supporting live systems, investigating production incidents, and improving operational stability.
- Familiarity with CI/CD pipelines and deployment troubleshooting (GitHub Actions and Azure-hosted environments experience preferred).
- Comfortable using AI-assisted engineering tools (e.g., GitHub Copilot) responsibly for code scaffolding, refactoring assistance, test generation, and documentation.
- Basic SRE concepts: reliability, monitoring, alerting awareness, operational risk reduction.
